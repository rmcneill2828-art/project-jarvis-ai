# 7. Governance Boundaries

ESR-0007 initialisation does not:

- approve an implementation package;
- create or modify AIEMS standards;
- create an Architecture Decision Record;
- modify controlled registers;
- update programme status;
- authorise product capability expansion.

Approved implementation scope shall be established only through a future authorised Engineering Implementation Brief or equivalent approved work package.

# 8. OSE And Repository Practice
